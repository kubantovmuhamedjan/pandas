# Pandas Data Manipulation & Analysis 🐼

Welcome to my **Pandas** practice repository. Here, I document my journey in learning data structures (Series & DataFrames), handling missing data, and performing exploratory data analysis (EDA).

## 📊 Core Concepts Practiced
* **Data Loading & Storage:** Reading and writing structured data using `.csv`, `.xlsx`, and performance-optimized formats like `.parquet`.
* **Data Inspection:** Utilizing `.head()`, `.info()`, `.describe()`, and memory usage tracking.
* **Data Cleaning:** Handling missing values (`.dropna()`, `.fillna()`), renaming columns, and changing data types.
* **Filtering & Selection:** Advanced indexing using `.loc[]`, `.iloc[]`, and conditional filtering.
* **Aggregation:** Grouping and summarizing datasets with `.groupby()` and `.agg()`.
* **Merging Data:** Combining multiple datasets using `.merge()` and `.concat()`.
